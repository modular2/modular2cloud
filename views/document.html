简单易用，快速接入

Modular2Cloud 是为modular2 模块化电脑开发的一个轻量级物联网平台.它秉承了modular2 "易学好用，快速部署"的设计理念，让用户可以快速地测试和部署IoT应用项目。技术上它具有如下特点：

采用了IPSO 信息模型

这样做的好处在于平台和应用无关，无论设备是一个电表网关，还是一个设备检测系统，都可以方便地接入平台，不需要在平台上设置大量与应用相关的参数。

支持多种协议和接入方式

支持中国移动的Onenet，上海国动LoRA接入网，Arm 物联网服务pelion，支持MQTT，CoAP/UDP ,HTTP 协议。

RESTFull API

提供了完善的web API 供用户开发与应用有关的特定APP或者应用软件
